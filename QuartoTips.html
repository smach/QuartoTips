Filter an Arquero table to exactly match a viewof Inputs filter unless filter value is "All"

filteredReportsTable = reportsTable
  .params({ myFilter })
  .filter((d) =>
    myFilter !== "All" ? d.Source === myFilter : true)

Filter an Arquero Table by a viewof Inputs filter via regular expression, unless filter value is "All" 

filteredReportsTable = reportsTable
  .params({ myFilter })
  .filter((d) =>
    myFilter !== "All" ? op.match(d.myColumn, myFilter) : true)

Filter an Arquero table based on values of a multi-select Inputs.select and have all values display if nothing is selected yet

filteredReportsTable = reportsTable
  .params({ sourceFilter})
   .filter(aq.escape((d) => ( sourceFilter[0] !== "All" && sourceFilter.length !== 0)  ? sourceFilter.includes(d.myColumn) : true) )
